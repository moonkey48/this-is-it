# Privacy Policy

Context Capture does not collect, transmit, sell, or share user data.

When the user clicks the extension button, Context Capture temporarily reads the active Chrome tab so the user can drag-select part of the page. The selected page context is converted to Markdown and copied to the user's clipboard locally in the browser.

No page content, URLs, clipboard contents, browsing history, account information, analytics events, or identifiers are sent to any server.

The extension uses these Chrome permissions:

- `activeTab`: temporarily access the current tab only after the user invokes the extension.
- `scripting`: inject the selection UI into the current tab after user action.
- `clipboardWrite`: copy the captured Markdown to the user's clipboard.
